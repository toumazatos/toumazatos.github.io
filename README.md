# toumazatos.github.io

Static single-page personal academic website for GitHub Pages.

## Files

- `index.html` — main homepage
- `styles.css` — stylesheet
- `assets/img/IMG_0116.JPG` — portrait image
- `toumazatos_cv.pdf` — CV linked from homepage

## Deploy on GitHub Pages

For a repository named `toumazatos.github.io`, just place these files in the root of the default branch, usually `main`.

Then in GitHub:
1. Go to **Settings**
2. Open **Pages**
3. Under **Build and deployment**, choose **Deploy from a branch**
4. Select the `main` branch and the `/ (root)` folder
5. Save

Your site should then be served from:

`https://toumazatos.github.io/`

## Notes

- This version does not use Jekyll.
- Old Jekyll files such as `_config.yml`, `Gemfile`, markdown pages, and theme files are not needed.
- You can edit the content directly inside `index.html`.
